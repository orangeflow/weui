# 手機號輸入頁面 - HTML 實現

這是根據 Figma 設計稿實現的完整 HTML 版本，包含所有交互功能。

## 文件結構

```
figma-playground/
├── index.html          # 主 HTML 文件
├── styles.css          # 樣式文件
├── script.js           # 交互邏輯
└── README-IMPLEMENTATION.md
```

## 功能特性

### ✅ 已實現的交互功能

1. **返回按鈕**
   - 點擊有視覺反饋
   - 顯示提示信息

2. **手機號輸入框**
   - 自動格式化（每4位數字加空格）
   - 只允許輸入數字
   - 聚焦時邊框高亮
   - 輸入時顯示清除按鈕
   - 清除按鈕可點擊清除內容

3. **協議複選框**
   - 點擊切換狀態
   - 視覺反饋動畫
   - 影響創建賬戶按鈕狀態

4. **創建賬戶按鈕**
   - 根據輸入和協議狀態自動啟用/禁用
   - 點擊有動畫效果
   - 驗證輸入有效性
   - 模擬 API 調用
   - 顯示賬戶已存在模態框

5. **模態框（賬戶已存在）**
   - 平滑的滑入動畫
   - 遮罩層點擊關閉
   - 關閉按鈕可點擊
   - 立即登錄按鈕可點擊

6. **視覺反饋**
   - 所有按鈕都有 hover 和 active 狀態
   - 點擊動畫效果
   - 輸入框聚焦效果
   - 模態框淡入淡出動畫

## 使用方法

### 直接在瀏覽器中打開

1. 雙擊 `index.html` 文件
2. 或在瀏覽器中打開該文件

### 使用本地服務器（推薦）

```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx http-server

# 然後訪問
# http://localhost:8000
```

## 設計規格

### 尺寸
- 頁面寬度：750px
- 頁面高度：1624px
- 移動端適配：響應式設計

### 顏色
- 主色：`#165dff` (藍色)
- 文字主色：`rgba(0, 0, 0, 0.9)`
- 文字次色：`rgba(0, 0, 0, 0.6)`

  - 按鈕：32px

### 間距

- 中間距：24px

- 大間距：48px

## 交互說明

### 手機號驗證
- 最少需要 11 位數字
- 自動格式化顯示
- 輸入時實時驗證

### 按鈕狀態
- **創建賬戶按鈕**：
  - 禁用：手機號無效或未同意協議
  - 啟用：手機號有效且已同意協議

### 模態框觸發
- 點擊創建賬戶按鈕後
- 模擬 API 返回賬戶已存在的情況
- 顯示登錄提示模態框

## 調試功能

在瀏覽器控制台中可以使用以下調試函數：

```javascript
// 顯示模態框
window.debugFigma.showModal();

// 關閉模態框
window.debugFigma.closeModal();

// 驗證手機號
window.debugFigma.validatePhone('185 7550 3480');

// 更新按鈕狀態
window.debugFigma.updateButton();
```
