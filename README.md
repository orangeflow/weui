# Figma Playground

这是一个包含多个项目的综合仓库，每个项目都有对应的文件夹。

## 项目结构

### 📚 weui-docs/
MimicUI 设计体系文档和工具
- 色彩体系 Token 表（colorToken.html）
- 设计 Token 表（designTokenTable.html）
- 文字样式 Token 表（typographyTokenTable.html）
- 相关的 CSS、JS 文件和文档

### 📱 mobile-app/
Web3 Wallet 移动应用 HTML 页面
- 注册/登录流程（app.html）
- 输入手机号页面（index.html）
- 相关的样式和脚本文件

### 🧧 redpacket-app/
红包应用 HTML 页面
- 红包封面选择页面（redpacket.html）
- 支付成功页面（success.html）
- 相关的样式和脚本文件

### ⚛️ nextjs-app/
Next.js 支付应用
- Next.js 应用代码（app/）
- NFC 支付相关组件（components/）
- 支付界面和交互逻辑

### 🧧 redpacket-app/
红包应用项目
- 红包 HTML 页面（redpacket.html）
- 红包 React 组件（components/RedPacket/）
- 完整的红包功能实现

### 💳 payment-app/
支付应用 HTML 页面
- 支付成功页面（success.html）
- 相关的样式和脚本文件

### 📖 docs/
项目文档
- 各种 README 和指南文档
- MimicUI 学习资料
- 实施指南

## 快速开始

### MimicUI 文档
```bash
cd weui-docs
# 打开 index.html 查看文档入口
```

### 移动应用
```bash
cd mobile-app
# 打开 index.html 或 app.html
```

### Next.js 应用
```bash
cd nextjs-app
npm install
npm run dev
```

## 项目说明

每个子项目都有独立的文档和说明，请查看对应文件夹中的 README 文件。
