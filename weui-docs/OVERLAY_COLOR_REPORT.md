## 颜色叠加计算公式

```
result = baseColor × (1 - alpha) + overlayColor × alpha
```

### 示例计算

**次要背景 (Light)**: `#F7F7F7`
- 基础色: `#FFFFFF` (255, 255, 255)
- 遮罩色: `#000000` (0, 0, 0)
- 透明度: 3% (0.03)

计算：
- R = 255 × (1 - 0.03) + 0 × 0.03 = 247.35 ≈ 247 = `F7`
- G = 255 × (1 - 0.03) + 0 × 0.03 = 247.35 ≈ 247 = `F7`
- B = 255 × (1 - 0.03) + 0 × 0.03 = 247.35 ≈ 247 = `F7`
- 结果: `#F7F7F7` ✅

## 重要说明

⚠️ **注意**：虽然计算结果接近，但这**不意味着这些颜色一定是通过叠加产生的**。

在实际设计中：
1. **设计工具（如 Figma）**：可能直接定义这些颜色值
2. **CSS 实现**：可能使用纯色值，也可能使用叠加层
3. **品牌色变体**：通常使用独立定义的纯色，以确保视觉精确度

## 建议

1. **遮罩层颜色**：明确标注为 `rgba()` 格式，表示它们是叠加层
2. **纯色变体**：如果使用独立定义，应在文档中说明
3. **实现方式**：在代码注释中说明是使用纯色还是叠加层
