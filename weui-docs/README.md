# MimicUI 设计体系文档

MimicUI 设计体系的完整文档和工具集合。

## 文件说明

### 主要文档页面
- **index.html** - 文档入口页面
- **colorToken.html** - 色彩体系 Token 表，包含白天和暗黑模式对比
- **designTokenTable.html** - 完整的设计 Token 表
- **typographyTokenTable.html** - 文字样式 Token 表

### 示例页面
- **redpacket.html** - 红包封面选择页面示例
- **success.html** - 支付成功页面示例

### 工具脚本
- **check_missing_tokens.js** - 检查缺失的 Token
- **check_overlay_colors.js** - 检查遮罩颜色
- **verify_weui_compliance.js** - 验证 MimicUI 合规性

### 文档
- **README.md** - 本目录说明
- **WEUI_OFFICIAL_REFERENCE.md** - MimicUI 官方参考文档
- **DEPLOY_GUIDE.md** - 部署指南
- **CHECK_TOKENS_README.md** - Token 检查说明

## 使用方法

直接在浏览器中打开 `index.html` 即可查看文档入口。
